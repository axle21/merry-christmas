# Merry Christmas 2025 Animation

This is a festive, interactive Christmas animation built with HTML, CSS, and JavaScript.

## How to Customize the Name

You can dynamically change the family name displayed on the page by modifying the URL. No code changes are required!

Simply add `?name=Your Name` to the end of the URL in your browser's address bar.

### Examples:

- **Default:** `index.html`  
  *Displays: "Rodriguez family"*

- **Custom Name:** `index.html?name=Smith Family`  
  *Displays: "Smith Family"*

- **Custom Message:** `index.html?name=Happy Holidays!`  
  *Displays: "Happy Holidays!"*

## Features

- **Interactive Snowfall:** Move your mouse left and right to control the wind direction.
- **Parallax Effect:** Background elements move slightly with your mouse for a 3D feel.
- **Twinkling Stars:** A dynamically generated starry background.
- **Responsive Design:** Works on different screen sizes.
